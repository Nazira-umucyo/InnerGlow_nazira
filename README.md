# 🧠 MindTrack – Mental Health & Mood Tracker

Welcome to **MindTrack**, a simple mental health web application that provides motivational quotes, mental health tips, and a safe space for self-reflection.

🚀 Features

- 🔐 User Login & Signup (localStorage)
- 🧠 Mood Tracking System
- 💡 Personalized Mental Health Tips
- 💬 Motivational Quotes (multiple + "New Quote" feature)
- 🌟 Daily Affirmations (API)
- 🤓 Fun Facts (API)
- 😂 Jokes (API)
- 📝 Personal Notes (stored locally)

🧩 System Overview
Actor: User
User Actions:
Login / Signup
Select mood
Get tips and quotes
Generate affirmation, joke, and fun facts
Save notes
System Actions:
Store user data (localStorage)
Fetch API data
Display dynamic responses
